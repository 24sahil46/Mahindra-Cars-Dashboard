# Mahindra Vehicle Sales Performance Dashboard 🚙📊

## Overview
This project is a comprehensive **Power BI Dashboard** designed to analyze the sales data of various Mahindra & Mahindra vehicle models. It transforms raw sales data into interactive visual reports, enabling stakeholders to quickly identify product performance, market trends, and key business metrics across different SUV and EV lines.

## Data Sources
The analysis is built on detailed sales data categorized by popular Mahindra models. The underlying datasets include:
* Thar
* Thar Roxx
* XUV400 EV
* XUV700
* Scorpio
* Scorpio N

## Tools & Technologies Used
* **Power BI:** Data modeling, DAX (Data Analysis Expressions), and interactive data visualization.
* **Power Query:** Data cleaning, transformation, and shaping.
* **Excel / CSV:** Raw data storage and source material.

## Key Features & Insights
* **Interactive Visualizations:** Dynamic slicers to filter data by specific vehicle models, regions, or timeframes.
* **Model Comparison:** Side-by-side performance tracking of flagship ICE (Internal Combustion Engine) models like the Scorpio N, XUV700, and the Thar family.
* **EV Tracking:** Dedicated metrics for the XUV400 EV to monitor electric vehicle market penetration and sales trends.
* **Sales KPIs:** Clear, high-level indicators for total sales volume, revenue estimates, and growth trajectories.

## Repository Structure
* `Dashboard.pbix`: The core Power BI project file containing the data model, relationships, and visual dashboard.
* `Mahindra_Sales_Data.xlsx - *.csv`: The source datasets separated by individual car models.
* `image_24171c.jpg`: A preview screenshot showcasing the UI and layout of the final dashboard.

## How to Run the Project locally
1. Clone this repository or download the ZIP file to your local machine.
2. Ensure you have the latest version of [Power BI Desktop](https://powerbi.microsoft.com/desktop/) installed.
3. Open the `Dashboard.pbix` file.
4. *(Optional)* If the data source links break, open **Power Query Editor**, go to Data Source Settings, and update the file paths to point to the folder where you saved the `.csv` files.
5. Interact with the dashboard to explore the insights!
